# Project Overview

This repository contains my Digital Literacy Project for the course CSE0001. The objective of this project is to develop essential digital skills required in academic and professional environments, including online presence, communication, collaboration tools, and cyber safety awareness.

# Detailed Task Description

## Task 1: Digital Literacy Awareness Infographic

In this task, I created a visual infographic using an online design tool (such as Canva). The infographic explains the concept of digital literacy, its importance, and key aspects like safe internet practices, digital tools for students, and maintaining a professional online presence. This task helped me understand how to present information in a clear and visually appealing way.

## Task 2: Student Digital Portfolio

In this task, I created my professional presence on platforms like GitHub, LinkedIn, and Kaggle. I set up my profiles with basic academic details and created a GitHub profile README. This task helped me understand the importance of building an online portfolio and how it can be useful for internships and career opportunities.

## Task 3: Coding & Collaboration Platforms

For this task, I explored HackerRank for coding practice and solved a beginner-level problem. I also created a Digital Literacy Awareness Quiz using Google Forms and analyzed responses using Google Sheets. This task improved my coding skills and taught me how to use online tools for collaboration and data collection.

## Task 4: Email Etiquette & Communication

In this task, I drafted two professional emails—one to a professor requesting an extension and another to an internship coordinator. I also created a checklist of Do’s and Don’ts for responsible social media usage. This task helped me understand the importance of clear, respectful, and professional communication.

## Task 5: Cybercrime Awareness

In this task, I studied cybercrime scenarios such as phishing and online fraud. I created a case study explaining how such crimes occur and a prevention checklist with safety tips. This task increased my awareness of online threats and taught me how to stay safe on the internet.
